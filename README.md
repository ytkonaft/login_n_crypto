**1. Run command for install dependencies**

```sh
$ yarn install
```

**2. Create .env file**

**3. Run node server**

```sh
$ yarn run-server
```

**4. Start webpack-dev-server**
Project is running on: http://localhost:3009

```sh
$ yarn dev
```

**5. Build the project**

```sh
$ yarn build
```

**Run test**

```sh
$ yarn test

$ yarn test-watch

$ yarn test-coverage
```
